How it works

Game state

Starting from a state of the game

Simulate

The AI traverses the game-tree based on its current understanding of strong play

Reward signal

Once the simulation is complete, the result of the game is used as a reward (or punishment) to train the neural network so that it learns from its good play, and from its mistakes

Implementation

Once the network is trained it is used as a policy generator and position evaluator in a tree-search algorithm such has MiniMax or Monte Carlo Tree Search
